Detailed Description

Convex Piecewise.

This class is a very simple convex piecewise linear function. The argument of the function is the expression. Between early_date and late_date, the value of the function is 0. Before early date, it is affine and the cost is early_cost * (early_date - x). After late_date, the cost is late_cost * (x - late_date).
