Detailed Description

Lin Kernighan.

For each variable in vars, stores the 'size' pairs(i,j) with the smallest value according to evaluator, where i is the index of the variable in vars and j is in the domain of the variable. Note that the resulting pairs are sorted. Works in O(size) per variable on average (same approach as qsort)
