void operations_research::RoutingModel::AddDisjunction ( const std::vector< int64 > &  nodes,
int64  penalty 
)

Adds a penalized disjunction constraint on the nodes: at most one of the nodes is active; if none are active a penalty cost is applied (this cost is added to the global cost function).

This is equivalent to adding the constraint: p + Sum(i)active[i] == 1, where p is a boolean variable and the following cost to the cost function: p * penalty. "penalty" must be positive.
