Detailed Description

This is based on Petr Vilim (public) PhD work.

All names comes from his work. See http://vilim.eu/petr. A theta-tree is a container for a set of intervals supporting the following operations: Insertions and deletion in O(log size_), with size_ the maximal number of tasks the tree may contain; Querying the following quantity in O(1): Max_{subset S of the set of contained intervals} ( Min_{i in S}(i.StartMin) + Sum_{i in S}(i.DurationMin) )
