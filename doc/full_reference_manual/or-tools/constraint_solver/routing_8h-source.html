00014 // Generic routing problem (TSP, VRP) modeller and solver.
00015 // Costs are specified by filling the cost matrix with SetCost().
00016 // Dimension and dimension constraints such as rank, capacity, time or distance
00017 // dimensions are specified using one of the AddDimensionXXX() methods;
00018 // corresponding variables can be retrieved using CumulVar().
00019 // The depot (starting and ending node for paths) is set with SetDepot; by
00020 // default it is set to node 0.
00021 // Solve() solves the routing problem and returns the corresponding assignment.
00022 // Optionally Solve can take an assignment that will be used to build a first
00023 // solution. The model is "closed" after the first call to Solve() and cannot
00024 // be modified afterwards; however Solve can be called more than once.
