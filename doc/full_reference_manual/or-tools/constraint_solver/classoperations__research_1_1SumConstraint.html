Detailed Description

Sum Array.

Some of these optimizations here are described in: "Bounds consistency techniques for long linear constraints". In Workshop on Techniques for Implementing Constraint Programming Systems (TRICS), a workshop of CP 2002, N. Beldiceanu, W. Harvey, Martin Henz, Francois Laburthe, Eric Monfroy, Tobias Müller, Laurent Perron and Christian Schulte editors, pages 39–46, 2002. SumConstraint This constraint implements sum(vars) == sum_var.
