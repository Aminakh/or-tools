Detailed Description

In all the following, the term 'update' means 'a potential new start min for a task'.

The edge-finding algorithm is in two phase: one compute potential new start mins, the other detects whether they are applicable or not for each task. Collection of all updates (i.e., potential new start mins) for a given value of the demand.
