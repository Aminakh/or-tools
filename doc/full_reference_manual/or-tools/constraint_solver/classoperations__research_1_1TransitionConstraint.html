Detailed Description

Deterministic Finite Automaton.

This constraint implements a finite automaton when transitions are the values of the variables in the array. that is state[i+1] = transition[var[i]][state[i]] if (state[i], var[i], state[i+1]) in the transition table. There is only one possible transition for a state/value pair.
