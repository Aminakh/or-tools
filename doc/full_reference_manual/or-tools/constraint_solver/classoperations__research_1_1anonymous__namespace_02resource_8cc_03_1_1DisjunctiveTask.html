Detailed Description

A DisjunctiveTask is a non-preemptive task sharing a disjunctive resource.

That is, it corresponds to an interval, and this interval cannot overlap with any other interval of a DisjunctiveTask sharing the same resource.

As this is just a pointer, copying is allowed. Assignment is not, since the pointer is const.
