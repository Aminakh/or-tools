template<int NumBytes>
int64 operations_research::PackedArray< NumBytes >::Value ( int64  index  )  const [inline]

Returns the value stored at index.

This assumes that the underlying architecture is little-endian. In particular, we select the value to return from the low-address NumBytes bytes of the int64 beginning at the location determined by the given index. To extract those NumBytes from the int64 loaded from memory, we choose the bytes of low arithmetic significance. Code for a big-endian system would need to choose the bytes of high arithmetic significance.
