NodeIndex operations_research::EbertGraph< NodeIndexSize, ArcIndexSize >::NextNode ( const NodeIndex  node  )  const [inline, private]

Returns the node following the argument in the graph.

Returns kNilNode (= end) if the range of nodes has been exhausted. It is called by NodeIterator::Next() and as such does not expect do be passed an argument equal to kNilNode. This is why the return line is simplified from return ( node == kNilNode || node >= num_nodes_) ? kNilNode : node + 1; to return node >= num_nodes_ ? kNilNode : node + 1;
