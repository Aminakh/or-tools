00020 // What is Linear Programming ?
00021 //
00022 //   In mathematics, linear programming (LP) is a technique for optimization of
00023 //   a linear objective function, subject to linear equality and linear
00024 //   inequality constraints. Informally, linear programming determines the way
00025 //   to achieve the best outcome (such as maximum profit or lowest cost) in a
00026 //   given mathematical model and given some list of requirements represented
00027 //   as linear equations. The linear programming problem was first shown to be
00028 //   solvable in polynomial time by Leonid Khachiyan in 1979, but a larger
00029 //   theoretical and practical breakthrough in the field came in 1984 when
00030 //   Narendra Karmarkar introduced a new interior point method for solving
00031 //   linear programming problems.
00032 //
00033 // -----------------------------------
00034 // What is Mixed Integer Programming ?
00035 //
00036 //   If only some of the unknown variables are required to be integers, then
00037 //   the problem is called a mixed integer programming (MIP) problem. These are
00038 //   generally also NP-hard.
00039 //
00040 // -----------------------------------
00041 // Check Wikipedia for more detail:
00042 //
00043 //   http://en.wikipedia.org/wiki/Linear_programming
00044 //
00045 // -----------------------------------
00046 // Example of a Linear Programming:
00047 //
00048 //   mimimize:
00049 //     f1*x1+f2*x2+...fn*xn
00050 //   subject to:
00051 //     a1*x1+a2*x2+...an*xn >= k1
00052 //     b1*x1+b2*x2+...bn*xn <= k2
00053 //     c1*x1+c2*x2+...cn*xn =  k3
00054 //     ......
00055 //     u1 <= x1 <= v1
00056 //     u2 <= x2 <= v2
00057 //     .....
00058 //
00059 //  As can be seen, Linear Programming has:
00060 //    1) linear objective function
00061 //    2) linear constraint
00062 //
00063 //  Note:
00064 //    The objective function is linear and convex.
00065 //    The constraints form a convex space if feasible.
00066 //
00067 // -----------------------------------
