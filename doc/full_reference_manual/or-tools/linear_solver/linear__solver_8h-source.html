00020 // -----------------------------------
00021 // What is Linear Programming ?
00022 //
00023 //   In mathematics, linear programming (LP) is a technique for optimization of
00024 //   a linear objective function, subject to linear equality and linear
00025 //   inequality constraints. Informally, linear programming determines the way
00026 //   to achieve the best outcome (such as maximum profit or lowest cost) in a
00027 //   given mathematical model and given some list of requirements represented
00028 //   as linear equations. The linear programming problem was first shown to be
00029 //   solvable in polynomial time by Leonid Khachiyan in 1979, but a larger
00030 //   theoretical and practical breakthrough in the field came in 1984 when
00031 //   Narendra Karmarkar introduced a new interior point method for solving
00032 //   linear programming problems.
00033 //
00034 // -----------------------------------
00035 // What is Mixed Integer Programming ?
00036 //
00037 //   If only some of the unknown variables are required to be integers, then
00038 //   the problem is called a mixed integer programming (MIP) problem. These are
00039 //   generally also NP-hard.
00040 //
00041 // -----------------------------------
00042 // Check Wikipedia for more detail:
00043 //
00044 //   http://en.wikipedia.org/wiki/Linear_programming
00045 //
00046 // -----------------------------------
00047 // Example of a Linear Programming:
00048 //
00049 //   mimimize:
00050 //     f1*x1+f2*x2+...fn*xn
00051 //   subject to:
00052 //     a1*x1+a2*x2+...an*xn >= k1
00053 //     b1*x1+b2*x2+...bn*xn <= k2
00054 //     c1*x1+c2*x2+...cn*xn =  k3
00055 //     ......
00056 //     u1 <= x1 <= v1
00057 //     u2 <= x2 <= v2
00058 //     .....
00059 //
00060 //  As can be seen, Linear Programming has:
00061 //    1) linear objective function
00062 //    2) linear constraint
00063 //
00064 //  Note:
00065 //    The objective function is linear and convex.
00066 //    The constraints form a convex space if feasible.
00067 //
00068 // -----------------------------------
